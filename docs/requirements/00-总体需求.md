# AI助手资源分享平台 - 总体需求

## 项目概述

构建一个AI助手资源分享平台，用户可以上传和下载MCP（Model Context Protocol）配置、Skills技能包，并通过评论、收藏、排行榜等功能进行社区互动。

## 部署环境

- 云服务器：2核CPU，2G内存，50G SSD
- 个人小型云服务器，资源有限
- 需要在有限资源下保证系统稳定运行

## 核心功能需求

### 1. 用户认证系统
- 用户注册（用户名+密码）
- 用户登录
- 会话管理
- 用户登出
- 密码加密存储（bcrypt）

### 2. MCP管理
- 上传MCP配置文件（需要登录）
- 下载MCP文件（无需登录）
- 浏览MCP列表
- 搜索MCP
- 管理自己的MCP（更新、删除）
- 查看MCP详情（介绍、版本、作者、下载统计）

### 3. Skills管理
- 上传Skills技能包（需要登录）
- 下载Skills文件（无需登录）
- 浏览Skills列表
- 搜索Skills
- 管理自己的Skills（更新、删除）
- 查看Skills详情（介绍、版本、作者、下载统计）

### 4. 评论系统
- 对MCP和Skills发表评论
- 查看评论列表
- 编辑自己的评论
- 删除自己的评论
- 举报不当评论

### 5. 收藏系统
- 收藏MCP和Skills（需要登录）
- 取消收藏
- 查看收藏列表
- 显示收藏数量

### 6. 排行榜系统
- 下载排行榜（日榜、周榜、月榜、总榜）
- 分类排行榜（MCP、Skills）
- 显示排名、资源名称、作者、下载次数、收藏次数

### 7. 内容分享
- 发布AI使用心得文章
- 浏览文章列表
- 搜索文章
- 查看文章详情
- 管理自己的文章（编辑、删除）
- 对文章发表评论

### 8. AI工具下载区
- 浏览AI工具列表
- 按类别筛选工具
- 查看工具详情
- 提供工具下载链接
- 搜索工具
- 推荐热门工具

### 9. 文件安全
- 文件类型白名单验证
- 文件大小限制（MCP: 10MB, Skills: 50MB）
- 病毒扫描
- 恶意代码检测

## 访问控制

### 无需登录
- 浏览所有资源列表
- 搜索资源
- 查看资源详情
- 下载MCP和Skills文件
- 查看评论
- 查看排行榜
- 浏览文章
- 查看工具列表

### 需要登录
- 上传MCP和Skills
- 发表评论
- 收藏资源
- 管理自己的资源
- 发布文章

## 技术栈

### 前端
- React 18+
- TypeScript
- Tailwind CSS
- Vite（构建工具）

### 后端
- Node.js 18+
- Express
- TypeScript

### 数据库
- MySQL 8.0（主数据库）
- Redis（缓存和会话）

### 文件存储
- 本地文件系统（初期）
- 预留对象存储接口（扩展期）

### 部署
- Nginx（反向代理）
- PM2（进程管理）
- Let's Encrypt（HTTPS证书）

## 性能要求

- 页面加载时间 < 3秒
- API响应时间 < 500ms（P95）
- 支持50并发用户
- 内存使用率 < 80%
- CPU使用率 < 70%（平均）
- 磁盘使用率 < 80%

## 安全要求

- HTTPS加密传输
- 密码bcrypt加密存储
- JWT token认证
- 文件类型验证
- 文件大小限制
- 病毒扫描
- SQL注入防护
- XSS防护
- CSRF防护

## 数据库设计要点

### 主要数据表
- users（用户表）
- resources（资源表：MCP和Skills）
- comments（评论表）
- favorites（收藏表）
- articles（文章表）
- tools（工具表）
- download_logs（下载日志表）

### 索引设计
- 用户名唯一索引
- 资源名称索引
- 下载次数索引
- 全文搜索索引（name, description）

## 文件存储结构

```
/data/uploads/

  │   │   ├── uuid1.json
```
