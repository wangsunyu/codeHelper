## 优化建议

### 如果内存不足
1. 减少MySQL buffer pool（降到256M）
2. 减少Redis内存限制（降到50M）
3. 减少Node.js内存限制（降到300M）
4. 考虑升级到4G内存

### 如果磁盘不足
1. 清理旧的日志文件
2. 清理无效的上传文件
3. 迁移文件到对象存储
4. 考虑升级磁盘容量

### 如果性能不足
1. 优化数据库查询和索引
2. 增加Redis缓存使用
3. 启用Nginx缓存
4. 考虑使用CDN
5. 考虑升级服务器配置
