# 设计版本记录

本文档记录所有UI设计的版本变更历史。

## 版本号规则

采用语义化版本号：`v{major}.{minor}.{patch}`

- **Major（主版本）**：重大设计改版，风格或布局大幅变化
- **Minor（次版本）**：新增页面或重要功能
- **Patch（补丁版本）**：小修改、优化、bug修复

## v0.1.0 - 2026-03-18

### 新增

- **首页设计** (homepage.pen)
  - 顶部导航栏：Logo、导航菜单（首页、MCP配置、Skills技能、排行榜）、搜索框、登录按钮
  - 英雄区：主标题"发现和分享 AI 助手资源"、副标题、两个CTA按钮（浏览MCP配置、浏览Skills技能）
  - 特色资源区：3列资源卡片网格，展示Claude MCP Server、Web Scraper Skill、Database Helper
  - 热门排行榜：前3名资源展示，包含排名、标题、下载数
  - 底部栏：关于平台、资源、社区三栏链接和版权信息

- **登录页面设计** (auth.pen)
  - 左侧品牌区（50%宽度）：Logo、品牌标语、特性列表（海量资源、活跃社区、完全免费）
  - 右侧表单区（50%宽度）：登录表单、用户名密码输入框、记住我选项
  - 登录按钮、分隔线、注册链接、服务条款说明

### 设计风格

- **风格定位：** Scandinavian Minimal（斯堪的纳维亚简约风格）
- **主色调：** Sage Green (#7C9082) - 温暖的鼠尾草绿
- **背景色：** Warm Cream (#FAF8F5) - 温暖的奶油色
- **卡片背景：** Pure White (#FFFFFF)
- **字体系统：**
  - 标题：Fraunces（衬线字体，优雅精致）
  - 正文：Inter（无衬线字体，清晰易读）
  - 数据：IBM Plex Mono（等宽字体，技术感）
- **圆角：** 20px（卡片）、24px（按钮）、12px（输入框）、28px（搜索框）
- **间距：** 基于4px网格系统，主要使用16px、20px、32px、40px、80px

### 设计特点

1. **温暖有机：** 使用温暖的奶油色背景替代冷白色，营造舒适氛围
2. **优雅层次：** 衬线标题 + 无衬线正文，创建清晰的视觉层次
3. **自然配色：** 鼠尾草绿作为唯一强调色，源于自然，传达平静和真实感
4. **慷慨留白：** 充足的间距和呼吸空间，提升可读性
5. **柔和圆角：** 20px圆角创造友好亲和的容器感
6. **开发者友好：** 简洁专注的界面，适合技术用户
